# E_commers 

This is the **backend for the E-Commerce project** built using **Java Spring Boot**.  
It provides REST APIs for user authentication, product management, shopping cart, and order processing. The backend can be used with any frontend or API client like Postman.

## Features

- User registration and login (JWT Authentication)
- Role-based access (Admin/User)
- Product management (CRUD operations)
- Shopping cart management
- Place orders and view order history
- REST APIs only 
- Optional Spring Security (can be enabled/disabled)
- Database persistence using Spring Data JPA

## Technology Stack

- **Backend:** Java, Spring Boot, Spring MVC, Spring Data JPA  
- **Security:** JWT (Spring Security optional)  
- **Database:** H2 / MySQL (configurable)  
- **Build Tool:** Maven
